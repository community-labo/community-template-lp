# community-template-lp

AIサロン向けの静的ランディングページです。トップページ、下層ページ、過去LPをHTML単体で管理しています。

## フォルダ構成

```text
.
├── index.html              # トップページ
├── contact.html            # お問い合わせ
├── privacy.html            # プライバシーポリシー
├── terms.html              # 利用規約
├── tokushoho.html          # 特定商取引法に基づく表記
├── assets/                 # トップページ共通画像
├── lp1/                    # 過去LP 1
├── lp2/                    # 過去LP 2
└── scripts/                # 補助スクリプト
```

## テンプレート利用時の公開前チェック

配布テンプレートとして利用する場合、公開前に以下を必ず差し替えてください。

- サービス名、説明文、料金、CTA文言
- 決済URL、問い合わせメールアドレス
- 特定商取引法に基づく表記の事業者情報
- プライバシーポリシー内の運営者情報、取得データ、利用目的
- 利用規約内の提供者名、サービス内容、禁止事項
- Google Tag Managerやアクセス解析ID
- 画像、プロフィール、イベント情報

本番のAPIキー、Stripe秘密鍵、SMTP認証情報、顧客情報、会員情報などの秘密情報はHTMLやGit管理下のファイルに記載しないでください。セキュリティ方針は `SECURITY.md` を参照してください。

## 運用メモ

- 画像は `assets/` に集約し、ファイル名は用途が分かる英数字のケバブケースを推奨します。
- `.wrangler/` や `.env` はローカル環境用のためGit管理に含めません。
- ページを追加する場合は、既存の下層ページのヘッダー・フッター構成をコピーして、サービス名やCTAの表記を揃えてください。
- 大きなデザイン改修を行う場合は、`index.html` 内のCSSが肥大化しやすいため、将来的に `assets/css/` への分離を検討してください。
